    Advanced tips tricks antd design table

    When working with Ant Design (AntD) and particularly with its Table component, there are several advanced tips and tricks you can use to enhance your tables and improve their functionality. Here are some advanced techniques:

    Overview

    1. Custom Cell Rendering

    You can use the render function to create complex cell contents. For example, you can include custom components, conditionally format data, or add interactive elements.

    {
    title: 'Status',
    render: (text, record) => (
        <span style={{ color: record.status === 'Active' ? 'green' : 'red' }}>
        {text}
        </span>
    ),
}

    2. Expandable Rows

    You can add expandable rows to show additional details when a row is clicked.

    const expandable = {
    expandedRowRender: record => <p>{record.details}</p>,
    rowExpandable: record => record.details != null,
};

// In Table component
<Table expandable={expandable} columns={columns} dataSource={data} />

    3. Custom Sorting and Filtering

    Define custom sorting and filtering logic by using the sorter and filters properties in column definitions.

    {
    title: 'Age',
    dataIndex: 'age',

    sorter: (a, b) => a.age - b.age,

    sortDirections: ['descend', 'ascend'],

    filters: [
        { text: '20-30', value: '20-30' },
        { text: '30-40', value: '30-40' }
    ],

    onFilter: (value, record) => {
        const [min, max] = value.split('-').map(Number);
        return record.age >= min && record.age <= max;
    },
}

    4. Pagination and Infinite Scrolling

    Configure pagination with custom settings or implement infinite scrolling to handle large datasets.

    // Pagination configuration
const pagination = {
    pageSize: 10,
    total: data.length,
    onChange: page => console.log('Page:', page),
};

// Infinite scrolling
const handleScroll = e => {
    if (e.target.scrollHeight - e.target.scrollTop === e.target.clientHeight) {
        // Load more data
    }
};

<Table pagination={pagination} onScroll={handleScroll} columns={columns} dataSource={data} />

    5. Cell Merging

    You can merge cells in a table using the rowSpan and colSpan properties in the render function.

    {
    title: 'Name',
    render: (text, record, index) => ({
        children: text,
        props: {
            rowSpan: index === 0 ? 2 : 0,
            colSpan: 2,
        },
    }),
}

    6. Dynamic Column Visibility

    Allow users to toggle column visibility dynamically using state management.

    const [visibleColumns, setVisibleColumns] = useState(['name', 'age']);

const handleVisibilityChange = (key) => {
    setVisibleColumns(prev => prev.includes(key) ? prev.filter(col => col !== key) : [...prev, key]);
};

const columns = [
    {
        title: 'Name',
        dataIndex: 'name',
        key: 'name',
        ...visibleColumns.includes('name') && { visible: true },
    },
    {
        title: 'Age',
        dataIndex: 'age',
        key: 'age',
        ...visibleColumns.includes('age') && { visible: true },
    },
];

// Render toggle buttons for columns
{['name', 'age'].map(key => (
    <Button key={key} onClick={() => handleVisibilityChange(key)}>
        Toggle {key}
    </Button>
))}

// Table with dynamic columns
<Table columns={columns} dataSource={data} />

    7. Row Selection

    Customize row selection behavior to support multiple selection modes or custom checkboxes.

    const rowSelection = {
    onChange: (selectedRowKeys, selectedRows) => {
        console.log('Selected rows:', selectedRows);
    },
    getCheckboxProps: record => ({
        disabled: record.name === 'Disabled User', // Disable selection for specific rows
    }),
};

<Table rowSelection={rowSelection} columns={columns} dataSource={data} />

    8. Custom Filter Dropdowns

    Create custom filter dropdowns for more advanced filtering options.

    import { Menu, Dropdown, Input } from 'antd';
import { DownOutlined } from '@ant-design/icons';

const filterDropdown = ({
    setSelectedKeys,
    selectedKeys,
    confirm,
    clearFilters,
}) => (
    <div style={{ padding: 8 }}>
        <Input
            placeholder="Search..."
            value={selectedKeys[0]}
            onChange={e => setSelectedKeys(e.target.value ? [e.target.value] : [])}
            onPressEnter={() => confirm()}
            style={{ marginBottom: 8, display: 'block' }}
        />
        <Space>
            <Button
                type="primary"
                onClick={() => confirm()}
                icon={<SearchOutlined />}
                size="small"
                style={{ width: 90 }}
            >
                Search
            </Button>
            <Button onClick={() => clearFilters()} size="small" style={{ width: 90 }}>
                Reset
            </Button>
        </Space>
    </div>
);

{
    title: 'Name',
    dataIndex: 'name',
    filterDropdown: filterDropdown,
    onFilter: (value, record) => record.name.toLowerCase().includes(value.toLowerCase()),
}

    9. Tree Data

    Render hierarchical data with nested rows.

    const treeData = [
    {
        title: 'Parent 1',
        key: '0-0',
        children: [
            {
                title: 'Child 1',
                key: '0-0-0',
            },
            {
                title: 'Child 2',
                key: '0-0-1',
            },
        ],
    },
];

<Table treeData={treeData} columns={columns} />

    10. Custom Toolbar and Actions

    Add custom toolbar actions like exporting, printing, or custom buttons.

    const toolbar = (
    <div>
        <Button onClick={() => console.log('Export')}>Export</Button>
        <Button onClick={() => console.log('Print')}>Print</Button>
    </div>
);

// Render toolbar above the table
<>
    {toolbar}
    <Table columns={columns} dataSource={data} />
</>

    Using these advanced tips and tricks can help you leverage the full power of Ant Design’s Table component and tailor it to meet your specific needs and use cases.
