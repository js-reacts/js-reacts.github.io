    Intro

    Most developers use Redux, Recoil, Mobx, and other third-party libraries to manage the global state in large-scale applications.

    • Redux

      A JS library for predictable and maintainable global state management.

      Reference

    • Mobx

      MobX is a signal based, battle-tested library that makes state management simple and scalable by transparently applying functional reactive programming.

      Reference

    • Recoil

      A state management library for React.

      Reference

    • Zustand

      A small, fast, and scalable bearbones state management solution.

      Reference
